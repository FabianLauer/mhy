# options

## --mhy-help

```bash
# show usage details
mhy --mhy-help
```

## --mhy-debug

```bash
# show debug messages
mhy --mhy-debug
```

## --mhy-verbose

```bash
# show more info than usual
mhy --mhy-verbose
```

## --mhy-prod

Forces `process.env.NODE_ENV` to be `production`.

## --mhy-version

```bash
# show mhy's currently installed version
mhy --mhy-version
```
