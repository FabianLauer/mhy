# node

Run scripts in Node.JS.
- Babel + TypeScript support.
- Support for aliases.
- Default: `./src`
- Resolve both `.js` and `.ts` files.

## Default command
```bash
node ./src
```

## CLI Options
https://nodejs.org/api/cli.html

## Examples

### Default CLI
```bash
mhy node
```

### Run a specific file
```bash
mhy node file
```
